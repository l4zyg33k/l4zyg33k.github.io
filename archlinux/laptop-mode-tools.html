Laptop mode tools 사용하기

acpid 서비스 설치

pacin acpid
systemctl enable acpid.service
systemctl start acpid.service

laptop-mode-tooles 설치

yay -Syu laptop-mode-toole
systemctl enable laptop-mode.service
systemctl start laptop-mode.service
